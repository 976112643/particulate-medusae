Particulate Medusae
===========================================================

Built with [particulate.js](http://particulatejs.org) and
[three.js](http://threejs.org) by
[jpweeks](http://twitter.com/jpweeks).

Dependencies
-----------------------------------------------------------

You should have one path for each dependency.

```sh
which node npm grunt
```

Installing
-----------------------------------------------------------

```sh
cd <PATH_TO_PROJECT>
npm install
```

Running
-----------------------------------------------------------

```sh
cd <PATH_TO_PROJECT>
grunt server:8001
```
